📌 Introduction / परिचय (Full Details)

🔹 What is this Smart ID Card System?
This is an advanced Digital Identity Ecosystem where a single smart card replaces all physical documents. Instead of carrying Aadhaar, PAN, Voter ID, Driving Licence, Passport, and other certificates, citizens carry ONE SMART CARD that securely links to all their data.

🔹 Key Concept: "One Card for All" — The card itself contains NO sensitive data. It only stores a secure encrypted token. When scanned (via NFC), the system fetches the required data from a centralized secure cloud database based on permissions.

हिंदी में पूरी जानकारी: यह एक डिजिटल पहचान प्रणाली है जिसमें एक स्मार्ट कार्ड से आधार, पैन, वोटर आईडी, ड्राइविंग लाइसेंस आदि सभी दस्तावेज़ जुड़े होते हैं। कार्ड में सिर्फ एक टोकन (गुप्त कोड) होता है, असली डेटा सुरक्षित क्लाउड पर रहता है।
🔄 How the Card Works - Complete Process

📱 Step 1: Card Scan (NFC)

User taps the Smart Card on any NFC-enabled device (mobile, CSC machine, traffic police device). The reader reads the secure token from the card's chip.

NFC 13.56 MHz

☁️ Step 2: Cloud Fetch

The token is sent to the Secure Government Cloud Database. The system verifies the token and retrieves only the necessary information based on the requester's permission level.

AES-256 Encryption TLS 1.3

🔢 Step 3: T-Code Generation

For website form filling, user generates a temporary T-Code (valid 2-5 minutes) via mobile app. Entering this T-Code on any website auto-fills basic details (name, address, etc.).

OTP-like Time-limited

🔐 Step 4: Full Verification

For sensitive operations (banking, property registration), multi-factor authentication is required: Card + Fingerprint + OTP on registered mobile. Only then full data is shared.

Biometric 2FA/3FA

🔢 T-Code System - Complete Explanation

What is T-Code? A temporary, single-use or limited-time code that allows auto-filling forms without exposing your full data.

How to use:
1️⃣ Open Mobile App → "Generate T-Code"
2️⃣ Choose validity (2 min, 5 min, 10 min)
3️⃣ Select which fields to share
4️⃣ Enter 6-digit T-Code on any website form
5️⃣ Form auto-fills automatically → Submit

टी-कोड क्या है? यह एक अस्थायी कोड है जो बिना पूरा डेटा दिखाए वेबसाइट पर फॉर्म ऑटो-फिल करता है।
